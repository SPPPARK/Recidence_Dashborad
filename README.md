# Chicago Public Service Dashboard

Interactive dashboard analyzing Chicago neighborhood-level service accessibility.

## Overview
This project visualizes public service access across 77 Chicago neighborhoods using demographic and health data.
Originally designed for Azure SQL, but due to Azure for Students region restrictions, the data is hosted in Google Drive.

## Features
- ETL: JSON → Pandas → Google Drive (originally Azure SQL)
- Interactive dashboard in Python (Folium / Streamlit)
- Insights into demographic patterns and service inequality

## Tools & Technologies
`Python`, `SQL`, `Azure`, `Google Drive`, `Power BI`, `Tableau`, `Folium`, `Pandas`

## Results
The dashboard highlights underserved neighborhoods and supports data-driven policy recommendations.
